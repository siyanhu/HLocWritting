\section{Introduction}\label{sec:introduction}

The whole system of Tramway 3D reconstruction and localization is a multi-camera and visual localization framework that reconstructs 3D scenes from a synchronized 6-camera
rig mounted on a tramway vehicle, and then localize query images (captured by iPhone) within that reconstruction. The entire system is built upon the HLoc (Hierarchical Localization) framework and uses COLMAP as its core reconstruction engine.
